# data-visualization-studio
Interactive desktop-based data visualization studio built with PyQt5, Matplotlib, and Pandas for Excel/CSV analysis with advanced plotting, multi-file support, and GUI controls.# 📊 Excel Data Plotter (Intermediate Version)

## 🚀 Overview
This is an intermediate-level desktop application built using **Python, PyQt5, Pandas, and Matplotlib**.  
It allows users to load Excel/CSV files and visualize data using interactive plots with basic styling controls.

---

## ✨ Features
- Load Excel and CSV files
- Select X and Y columns for plotting
- Basic graph customization:
  - Line color
  - Line style
  - Line width
  - Markers
- Grid enable/disable
- Hold-on mode (overlay multiple plots)
- Simple GUI interface

---

## 🛠 Tech Stack
- Python
- PyQt5
- Pandas
- Matplotlib

---

## 📈 Purpose
This project demonstrates:
- GUI development with PyQt5
- Data visualization using Matplotlib
- Basic event-driven programming
- Data handling with Pandas
