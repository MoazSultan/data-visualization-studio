# 📊 Excel Plot Viewer (Advanced Version)

## 🚀 Overview
This is an advanced interactive data visualization tool built using **Python, PyQt5, Pandas, and Matplotlib**.  
It allows loading multiple Excel files and provides dynamic filtering, interactive plotting, and graph manipulation features.

---

## ✨ Features

### 📁 Data Handling
- Load multiple Excel files
- Store and manage multiple datasets
- Automatic file tracking system

### 🔍 Smart Filtering
- Search and filter columns dynamically
- Real-time column list updates

### 📊 Interactive Plotting
- Select X and Y variables
- Multi-line graph plotting
- Automatic legend updates

### 🖱️ Interactive Graph Features
- Right-click on plotted line
- Delete specific graph line
- Autoscale after deletion

### 🎛️ UI Features
- Clean PyQt5 interface
- File information table
- Column selection panel
- Clear all plots button
