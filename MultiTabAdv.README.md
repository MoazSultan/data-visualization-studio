# 📊 Multi-Tab Graph Viewer (Advanced Modular Version)

## 🚀 Overview
This is a professional-level desktop application built using **Python, PyQt5, Pandas, and Matplotlib**.  
It provides a multi-tab system for loading and visualizing datasets independently in each tab.

---

## ✨ Features

### 🪟 Multi-Tab System
- Each tab works as an independent graph workspace
- Load different datasets in separate tabs
- Easy switching between projects

### 📁 Data Handling
- Load Excel and CSV files
- Automatic column detection
- First column used as X-axis by default

### 📊 Data Visualization
- Plot multiple Y-axis variables
- Automatic legend generation
- Grid support

### 🎨 Graph Styling
- Line color selection
- Line style customization
- Line width control
- Marker options
- Marker size control

### 🖱️ Interactive Features
- Click-based graph interaction
- Line highlighting on selection
- Real-time updates
